This program is meant to help those in genetics courses that are struggling with the practices of DNA translation, transcription and reverse complementation. 

Pre-Requisites
  1- Python 3(https://www.python.org/downloads/)
  2- Biopython Library(http://biopython.org/)

Instructions
  1- Input the DNA sequence(Note: It must be a legitimate sequence. The program will notify you if it is not standardized, and will prompt you for a new input)
  2- Select the type of procedure that you wish to use. Translation, transcription and reverse complement are the available commands(Note: The commands are case-sensitive, so capitalization will have no effect on the program's functionality)
  3- View the results
  4- Choose to perform another procedure, or exit
